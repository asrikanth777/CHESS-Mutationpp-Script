
this is for using the mutation++ script
here is how to

1) 	clone this github repo into ur project folder
	it should be with all the exec, input, mesh, and output folders

2)	in the same project folder, open in terminal and type paraview
3)	then at the bottom right, press run script
	- this will bring up the current directory, ur project folder
	  navigate to CHESS-Mutationpp-script, and open the python file
	  it should have csv___.
	  
4)	once thats done, it will ask for an input
	- this is the output folder u want to analyze

	
5)	after that let it run, it takes a second to do it all
6)	click the GroupDatasets filter in the pipeline and then click on the
	open space in paraview,
	- this might take a second to render

thats it, when you go to the dropdown it should have have the data u want etc.:
-mole fractions
-entropy
-thermal conductivity

it can also be opened in spreadsheet view
